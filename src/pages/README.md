# src/pages

Páginas / rotas do painel — compõem `components` e chamam `core` para montar cada tela.

> Nota: ao fechar a stack em **Next.js (App Router)**, as rotas migram para `src/app/`.
> Mantido como `pages/` por ora conforme a nomenclatura combinada.

- Uma página por arquivo, um único objetivo de export.
